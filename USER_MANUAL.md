## 一、系統概述與角色權限

本系統為物流公司設計的日常作業管理平台，涵蓋送件記錄、薪資計算、車輛管理、請假、排班、加油回報等核心功能。

### 角色與權限對照表

| 角色代碼 | 顯示名稱 | 主要職責 |
|---------|---------|---------|
| `ADMIN` | 董事長 | 全系統管理，所有功能完整存取 |
| `MANAGER` | 執行長 | 營運與人事管理，可查看全公司數據 |
| `REGION_MANAGER` | 區經理 | 僅能管理所屬區域的成員 |
| `EMPLOYEE` | 員工 | 個人日常作業功能 |

> **注意：** 系統第一個註冊的帳號自動成為董事長（ADMIN）。

## 二、登入與帳號管理

### 登入（`/login`）

1. 輸入 Email 與密碼
2. 點擊「登入」，成功後自動進入首頁

### 註冊（`/register`）

- 若系統設定為「開放自行註冊」，任何人可自行建立帳號，角色預設為員工
- 若已關閉自行註冊，由董事長至「員工管理」頁面新增帳號

### 忘記密碼

請聯繫董事長或執行長，於「員工管理」→ 該員工 → 「重設密碼」，將新密碼告知本人。

## 三、首頁

**可使用角色：** 全體員工

首頁分為三個區塊：

### 公告欄

- 顯示公司公告，一則
- **董事長／執行長** 可點擊「編輯公告」修改內容後儲存
- 員工唯讀

### 行事曆

月曆整合三種資訊，以色彩 badge 區分：

| 顏色 | 意義 |
|------|------|
| 藍色 | 公司活動 |
| 橙色 | 當日請假人員 |
| 綠色 | 當日排班人員 |

**操作：**
- 點擊「上個月」「下個月」切換月份
- 點擊「本月」返回當月
- 點擊任一日期格子 → 開啟彈窗，查看當日排班、請假、活動明細
  - 董事長／執行長 可在彈窗內新增或刪除活動

### 我的排班快速欄（員工專屬）

首頁行事曆下方顯示個人排班：
- **今日排班：** 高亮標記「今天（區域名稱）」
- **近期排班：** 本月未來 4 筆排班，顯示日期與區域

## 四、核心日常作業

### 每日送件記錄（`/delivery`）

**可使用角色：** 全體

#### 填寫送件紀錄

| 欄位 | 說明 |
|------|------|
| 日期 | 預設今天，可改為過去任意日期 |
| 今日角色 | 無 ／ 貨車司機 ／ 貨車隨車人員 |
| 正物流件數 | 預設 0 |
| 逆物流件數 | 預設 0 |
| 備註 | 異常或補充說明（選填） |

填寫完畢後點擊「送出」。同一天若重新送出，系統自動覆蓋舊紀錄。

#### 歷史紀錄

頁面下方表格顯示個人歷史紀錄，可點選對應列重新編輯後送出。

#### 批次匯入（僅董事長）

1. 點擊「批次匯入」
2. 點「下載 Excel 範本」取得格式
3. 填寫資料（日期、員工姓名、正物流、逆物流、角色）
4. 上傳檔案 → 點「預覽」確認解析結果
5. 確認無誤後點「確認匯入」

---

### 車輛里程記錄（`/mileage`）

**可使用角色：** 全體

#### 填寫里程

每日填寫該車輛「今日結束里程」（累計 km），系統自動計算當日行駛里程。

| 欄位 | 說明 |
|------|------|
| 機車 | 從下拉選單選擇車牌（必填） |
| 今日結束里程 | 當天行駛結束後的累計里程數 |
| 當日行駛里程 | 自動計算：今日里程 − 上次里程 |
| 貨車 | 勾選「今日有使用貨車」後顯示，填寫方式同機車 |

> 輸入的結束里程必須 ≥ 上次紀錄，否則無法送出。

## 五、人事行政功能

### 我的薪資（`/salary/me`）

**可使用角色：** 全體

#### 查看薪資

頁面上方可選擇年份與月份，自動載入對應月份薪資。

**薪資組成說明：**

| 項目 | 說明 |
|------|------|
| 按件薪資 | 每日件數 × 當日適用單價 加總 |
| 貨車司機加給 | 司機天數 × 每日加給 |
| 貨車隨車人員加給 | 隨車天數 × 每日加給 |
| 職務加給 | 管理員設定的固定加給 |
| 激勵獎金 | 達成出勤與件數門檻時發放 |
| 油資補貼 | 當月已核准加油回報總額 |
| 扣款 | 管理員登記的扣款項目（紅字） |

**單價規則（按職稱判定）：**

| 職稱 | 當日件數條件 | 單價 |
|------|------------|------|
| 執行長 | 任何 | 30 元（固定） |
| 資深員工（高） | ≤ 100 件 | 25 元 |
| 資深員工（高） | > 100 件 | 28 元（全部件數） |
| 資深員工（低） | ≤ 100 件 | 23 元 |
| 資深員工（低） | > 100 件 | 26 元（全部件數） |
| 員工（高） | ≤ 100 件 | 25 元 |
| 員工（高） | > 100 件 | 28 元（全部件數） |
| 員工（低） | ≤ 100 件 | 23 元 |
| 員工（低） | > 100 件 | 26 元（全部件數） |
| 臨時工 | 任何 | 23 元（固定） |
| 特殊 | 任何 | 30 元（固定） |

> 件數 > 100 時，全部件數套用高單價，非分段計算。

### 請假申請（`/leaves`）

**可使用角色：** 全體

#### 申請請假

1. 選擇「請假日期」
2. 輸入原因（選填）
3. 點擊「送出申請」

#### 查看申請狀態

「我的請假紀錄」表格顯示所有申請，狀態說明：

| 狀態 | 顏色 | 說明 |
|------|------|------|
| 待審核 | 橙色 | 等待主管審批 |
| 已核准 | 綠色 | 核准，計入行事曆 |
| 已拒絕 | 灰色 | 未核准 |

待審核的申請可點「取消申請」撤回。

### 我的排班（`/my-schedule`）

**可使用角色：** 全體

- 月曆視圖顯示個人排班（綠色 badge 標記有排班的日期）
- 點擊日期查看詳細排班資訊（小區域名稱、備註）
- 頁面下方顯示本月排班統計與各區域次數

---

### 加油回報（`/fuel-report`）

**可使用角色：** 全體

#### 新增加油回報

| 欄位 | 說明 |
|------|------|
| 加油日期 | 預設今天 |
| 金額（元） | 必填，正整數 |
| 車輛類型 | 機車／貨車切換按鈕，選擇後下方車牌選單會依類型篩選 |
| 車牌 | 選填，從所選類型的車輛中選擇 |
| 備註 | 加油地點或說明（選填） |

填寫完畢點「送出回報」。歷史紀錄列表的車牌標籤會同時顯示「機車」或「貨車」字樣以便分辨。

#### 查看紀錄

- 頁面上方可切換月份
- 顯示本月已核准回報合計（將計入薪資）
- 待審核的回報可點「撤回」取消

#### 回報狀態

| 狀態 | 顏色 |
|------|------|
| 待審核 | 橙色 |
| 已核准 | 綠色 |
| 已駁回 | 紅色 |

## 六、排班系統（主管）

**頁面：** 排班管理（`/schedule`）  
**可使用角色：** 董事長、執行長、區經理

### 視圖切換

- **月曆視圖：** 點擊日期格子開啟排班編輯彈窗
- **列表視圖：** 按日期線性顯示所有排班，支援篩選搜尋

### 新增排班

**單人模式：**
1. 點擊目標日期
2. 選擇員工
3. 填寫小區域名稱（支援自動補全歷史名稱）
4. 選擇區域（選填）
5. 填寫備註（選填）
6. 點「新增」

**批次模式：**
1. 切換至「批次新增」
2. 多選員工
3. 填寫小區域名稱與備註
4. 點「批次新增」（同時為所有選取員工新增該日排班）

### 編輯／刪除排班

點擊現有排班記錄，修改後點「保存」；或點「刪除」移除。

## 七、油資審核（主管）

**頁面：** 油資審核（`/fuel-review`）  
**可使用角色：** 董事長、執行長、區經理

頁面分三個分頁：

### 待審核

列出所有待審核的加油回報（員工姓名、日期、金額、機車）。

- **核准：** 點「核准」→ 自動計入該員工當月薪資油資補貼
- **駁回：** 點「駁回」→ 輸入駁回原因後確認

### 歷史紀錄

顯示所有已核准／已駁回的回報，支援月份與員工篩選。

### 車輛油費

分「機車」「貨車」兩個區塊分別統計，每個區塊依車輛分組顯示當月油費合計，可展開查看每筆詳情。

## 八、薪資計算（管理員）

**頁面：** 薪資計算（`/admin/salary`）  
**可使用角色：** 董事長、執行長

### 查看全員薪資

選擇年月後，表格列出所有員工當月薪資概覽：  
員工姓名、職稱、出勤天數、總件數、按件薪資、司機加給、隨車加給、職務加給、激勵獎金、扣款、**實領薪資**。

### 展開個別員工詳情

點擊員工列「明細」按鈕：

#### 職稱覆蓋

系統依出勤與件數自動判定職稱，若需調整：
1. 點擊職稱下拉選單
2. 選擇正確職稱
3. 自動儲存（顯示「管理者手動覆蓋」標記）

#### 新增扣款

1. 輸入扣款原因
2. 輸入金額
3. 點「新增扣款」

#### 編輯每日送件紀錄

- **編輯：** 點「編輯」→ 修改正／逆物流件數 → 點「保存」
- **刪除：** 點「刪除」移除該日紀錄
- **新增：** 點「新增送件紀錄」→ 選擇日期、填件數 → 點「保存」

### 匯出

- **單一員工薪資單：** 點員工列「匯出薪資單」，下載 PDF
- **全公司薪資總表：** 頁面右上角「匯出 Excel」，含所有員工薪資摘要

## 九、管理者儀表板與報表

### 儀表板（`/admin`）

**可使用角色：** 董事長、執行長

顯示：
- **待處理事項提醒：** 未設定月度單價、上月未對帳、車輛保養逾期
- **月度統計：** 累計件數、預估薪資支出、預估總收入、預估毛利
- **今日快照（當月）：** 今日正／逆物流件數

---

### 每日營運總表（`/admin/daily-operations`）

按日列出：正物流、逆物流、薪資支出、營收、毛利、出勤人數、司機數、隨車數。  
表尾顯示月度平均與合計。

---

### 員工送件狀況（`/admin/delivery-status`）

選擇日期，顯示當日所有員工的送件情況（姓名、角色、今日角色、件數、備註）。

---

### 車輛狀況（`/admin/vehicle-status`）

顯示全部車輛的累計里程、保養狀態（正常／預警／逾期）、今日使用人員。

---

### 派遣紀錄（`/admin/dispatch`）

**可使用角色：** 董事長、執行長

查看指定日期全員派遣情況（誰開哪台車、誰是司機／隨車人員）。

**操作：**
- 修改今日角色：點角色下拉直接調整
- 編輯里程：點「編輯」→ 修改結束里程 → 「保存」
- 刪除里程記錄：點「刪除」確認後移除

## 十、員工管理

**頁面：** 員工管理（`/admin/employees`）  
**可使用角色：** 董事長（完整操作）、執行長（唯讀）

### 功能一覽

| 操作 | 步驟 |
|------|------|
| **變更角色** | 點選角色下拉 → 選擇新角色，自動儲存 |
| **設定特殊職稱** | 點選特殊職稱下拉 → 選擇「執行長」或「特殊」，自動儲存 |
| **停用帳號** | 切換狀態 toggle → 停用後帳號無法登入，紀錄保留 |
| **重設密碼** | 點「重設密碼」→ 輸入新密碼與確認密碼 → 確認 |
| **刪除帳號** | 需先至「員工紀錄管理」清空紀錄後，才可刪除 |

### 員工紀錄管理（`/admin/employees/:id/records`）

**可使用角色：** 董事長

顯示該員工所有歷史紀錄（送件、里程、請假、薪資調整、排班、加油回報）。

- **逐筆刪除：** 點各區塊的「刪除」按鈕
- **一鍵清空：** 點「清空所有紀錄」→ 輸入員工姓名確認 → 「清空」

> 需清空紀錄後，才能在「員工管理」頁刪除該帳號。

## 十一、車輛管理

**頁面：** 車輛管理（`/admin/vehicles`）  
**可使用角色：** 董事長、執行長

### 新增車輛

填寫車牌號碼、車型（機車／貨車）、備註、初始里程後點「新增車輛」。  
新車自動建立預設保養項目。

### 車輛詳情

點擊車輛記錄展開：

**保養項目管理**

每個保養項目顯示名稱、保養周期（km）、上次里程、下次應保養里程與狀態。

| 狀態 | 說明 |
|------|------|
| 綠色（正常） | 未到保養里程 |
| 橙色（預警） | 剩餘里程 < 週期 10%，建議準備 |
| 紅色（逾期） | 已超過應保養里程，需立即處理 |

**新增保養項目：** 填寫項目名稱與保養周期 → 點「新增項目」

**登記維修：**
1. 點「登記維修」
2. 選擇保養項目
3. 填寫當時里程與備註
4. 點「保存」→ 系統自動更新下次應保養里程

**使用歷史：** 顯示該車被使用的所有日期、使用人員、行駛里程。

## 十三、貨運行對帳

**頁面：** 貨運行 Excel 對帳（`/admin/reconciliation`）  
**可使用角色：** 董事長

### 操作步驟

1. 選擇對帳年月
2. 上傳貨運行提供的月結 Excel 檔案
3. 系統自動讀取件數與金額，與系統數據比對
4. 查看對帳結果：

| 差異顯示 | 意義 |
|---------|------|
| 紅字（+N） | 系統件數 < 貨運行件數（系統少計） |
| 綠字（−N） | 系統件數 > 貨運行件數（系統多計） |

5. 差異為零或在允許範圍內即完成對帳，系統自動儲存紀錄

頁面下方可查看歷史對帳紀錄。

## 十四、系統設定

**頁面：** 系統設定（`/admin/settings`）  
**可使用角色：** 董事長

### 薪資加給設定

- 貨車司機每日加給金額
- 貨車隨車人員每日加給金額

### 員工固定加給

為個別員工設定固定月度職務加給。

### 月度單價設定

每月需設定正物流與逆物流單價（元／件），薪資計算與營收估算皆依此月單價。

> **重要：** 每月月初記得設定該月單價，否則儀表板會顯示提醒。

### 開放自行註冊

Toggle 控制是否開放員工自行至 `/register` 建立帳號。

### 薪資計算公式設定

可調整以下參數（變更後即時生效於薪資計算）：

- **職稱判定門檻：** 資深員工／一般員工最低出勤天數、高／低職稱件數門檻
- **各職稱日薪單價：** 資深員工高、資深員工低、員工高、員工低、臨時工、特殊
- **激勵獎金階梯：** 兩階段，各設定出勤天數門檻、件數門檻、獎金金額
- **公式說明文字：** 顯示於薪資頁面，向員工解釋計算邏輯

## 十六、常見操作場景

### 場景一：月底發薪流程

1. 確認本月所有員工送件紀錄已填寫完整
2. 「系統設定」→ 確認本月正／逆物流單價已設定
3. 「油資審核」→ 核准所有待審核的加油回報
4. 「薪資計算」→ 選擇目標月份
5. 檢查各員工職稱判定，如有誤則手動覆蓋
6. 新增任何扣款項目
7. 點「匯出全公司薪資總表」（Excel）供財務核對
8. 依序點各員工「匯出薪資單」下載 PDF 發給員工

---

### 場景二：月度對帳流程

1. 收到貨運行月結 Excel
2. 「貨運行對帳」→ 選擇對帳年月
3. 上傳 Excel 檔案
4. 查看對帳差異
5. 若有差異，至「每日送件記錄」或「薪資計算」核對各日件數
6. 確認無誤後完成對帳，系統自動儲存紀錄

---

### 場景三：新員工入職

1. 董事長至「員工管理」→ 由「開放自行註冊」或直接新增帳號
2. 告知員工帳號與密碼
3. 至「區域管理」將員工加入對應區域
4. 員工登入後即可開始填寫每日送件與里程記錄

---

### 場景四：請假審批流程

1. 員工於「請假申請」送出申請
2. 執行長或董事長至「請假管理」查看待審核申請
3. 點「核准」或「拒絕」
4. 核准後該日自動顯示於首頁行事曆

---

### 場景五：員工離職帳號處理

1. 「員工管理」→ 先將帳號停用（防止登入）
2. 「員工紀錄管理」→ 選擇是否清空歷史紀錄
3. 若確認不需保留 → 點「清空所有紀錄」，輸入姓名確認
4. 返回「員工管理」→ 刪除帳號

## 十七、常見問題

**Q：為什麼職稱判定不符合預期？**  
A：至「系統設定」確認職稱判定門檻（出勤天數與日均件數）。若需調整，執行長可至「薪資計算」→ 展開員工詳情 → 手動覆蓋職稱。

**Q：油資補貼為何沒計入薪資？**  
A：確認加油回報已被「核准」（非待審核）。僅已核准的回報才會計入當月薪資。

**Q：里程紀錄填錯了怎麼辦？**  
A：執行長至「派遣紀錄」找到對應日期與車輛，點「編輯」修改里程數值，或點「刪除」後重新填寫。

**Q：車輛顯示保養逾期怎麼處理？**  
A：實際完成保養後，至「車輛管理」→ 點該車輛「登記維修」→ 選擇對應保養項目 → 填寫維修里程 → 儲存，系統即更新下次應保養里程。

**Q：員工帳號被停用，紀錄會消失嗎？**  
A：不會。停用只是禁止登入，所有歷史紀錄（送件、薪資等）完整保留。

**Q：行事曆上的排班 badge 為何只顯示 2 個人？**  
A：月曆格子空間有限，最多顯示 2 筆，超出顯示「+N 人」。點擊該格子可查看當日所有排班完整資訊。
