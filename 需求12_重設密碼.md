# 需求：ADMIN 幫員工重設密碼

## 背景
目前員工管理頁面只有姓名、Email、角色、特殊職稱、帳號狀態，沒有重設密碼的功能。
若員工忘記密碼，目前無法處理。

## 需求說明

### 前端（員工管理頁面）
- 每個員工列表旁新增「重設密碼」按鈕
- 點擊後跳出確認視窗，內含：
  - 新密碼輸入欄位
  - 確認密碼輸入欄位
  - 確認送出按鈕
- 兩次密碼不一致時顯示錯誤提示
- 送出成功後顯示「密碼已更新」提示

### 後端
- 新增 API：`PUT /api/employees/:id/password`
- 僅 ADMIN 可呼叫（需驗證 JWT 權限）
- 將新密碼 hash 後存入資料庫（使用現有的 bcrypt 方式）
