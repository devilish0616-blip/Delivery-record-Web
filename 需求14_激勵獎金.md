# 需求14：薪資計算新增激勵獎金欄位

## 計算規則

每月薪資計算時，依以下條件自動判定激勵獎金：

| 條件 | 獎金 |
|------|------|
| 出勤天數 ≥ 25天 **且** （正物流+逆物流合計）/ 出勤天數 > 60 | **+3,000 元** |
| 出勤天數 ≥ 25天 **且** （正物流+逆物流合計）/ 出勤天數 > 30 | **+1,500 元** |
| 其他情況 | 0 元 |

> 注意：兩個條件是 IF/ELSE 關係，日均件數 > 60 優先判定，不會同時加 3000+1500。

對應 Excel 公式參考：
```
=IFERROR(IF(AND(出勤天數>=25, 總件數/出勤天數>60), 3000,
         IF(AND(出勤天數>=25, 總件數/出勤天數>30), 1500, 0)), 0)
```

## 功能說明

### 後端（salaryService.ts）
- 在薪資計算邏輯中新增激勵獎金計算
- 計算方式：
  1. 取得該員工當月出勤天數
  2. 取得當月正物流+逆物流合計
  3. 計算日均件數 = 合計 / 出勤天數
  4. 依上表判定獎金金額
- 激勵獎金加入實領薪資總額

### 前端
- 薪資計算頁面的員工薪資明細中，新增「激勵獎金」欄位顯示
- 匯出的 Excel 薪資報表（總表＋個人明細）也要包含此欄位
