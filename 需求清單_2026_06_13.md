## 二、角色權限調整

### 2. 新增第三種角色：SUPERVISOR（主管）

| 角色 | 說明 |
|------|------|
| ADMIN | 可查看＋修改所有資料 |
| SUPERVISOR | 可查看所有資料，但**不能修改**（以下有例外） |
| EMPLOYEE | 只能看自己的資料＋填寫當日紀錄 |

**SUPERVISOR 例外可操作的項目：**
- ✅ 可以編輯車輛保養項目（標記已更換、填備註）
- ✅ 可以查看派遣紀錄

**SUPERVISOR 角色指派方式：**
- 由 ADMIN 在後台員工管理頁面手動指定（類似目前指定職稱的方式）

**SUPERVISOR 也需要填寫：**
- 每日正逆物流件數（他也要送貨）
- 每日車輛使用狀況（同員工流程）
- 每日司機／隨車人員角色

## 三、功能修改

### 3. ADMIN 也能填寫每日正逆物流
- 目前只有 EMPLOYEE 可以填寫每日送件記錄
- ADMIN 也需要能填寫自己的每日正逆物流（他也會送貨）
- 入口：讓 ADMIN 也能進入「每日送件記錄」填寫頁面

### 4. 員工記錄車輛使用狀況改版
目前的里程記錄方式需要改版，改為員工自己記錄當天車輛使用：

- **第一格（強制填）**：機車
  - 選擇機車車牌（下拉選單，只顯示啟用中的機車）
  - 填寫起始里程、結束里程
- **第二格（選填）**：貨車
  - 選擇貨車車牌（下拉選單，只顯示啟用中的貨車）
  - 填寫起始里程、結束里程
  - 沒有使用貨車可以不填

> 不需要管理者手動建立派遣紀錄，改由員工自己記錄

### 5. 每日司機／隨車人員記錄
- 員工在填寫每日記錄時，新增一個「今日角色」選單：
  - **無**（預設）
  - **司機**
  - **隨車人員**
- ADMIN 可以在後台修改所有人的今日角色

### 6. 派遣紀錄頁面改版
- 不再由管理者手動新增派遣
- 改為**統計查看頁面**：顯示「今天誰開哪輛車、誰是司機、誰是隨車人員」
- 資料來源：員工自己填寫的車輛使用紀錄＋今日角色

### 7. 車輛管理：後台可查每輛車使用歷史
- 管理者在車輛管理頁面，展開每輛車時可以看到：
  - 里程紀錄：誰、哪天、開了多少公里
  - 今日角色紀錄：誰填過司機／隨車
- 目的：讓管理者可以判斷這輛車是否有人使用過，再決定刪除或停用

### 8. 車輛車型可修改
- 目前編輯車輛時無法修改車型（機車／貨車）
- 改為可修改，但修改前需跳出確認視窗，提示：
  > 「變更車型將會重置預設保養項目，確定要修改嗎？」
- 確認後才執行變更

### 9. 管理者修改所有人正逆物流（確認現有功能）
- 根據 commit `3d5639e`，管理者在薪資計算頁面可展開員工每日明細並點擊「編輯」修改件數
- 請確認此功能是否正常運作，如有問題請修復

## 四、新功能

### 10. 一鍵生成單一員工薪資表
- 管理者在薪資計算頁面，可選擇：
  - 指定員工
  - 指定月份
- 點擊「匯出薪資單」後，產生該員工當月的薪資 Excel 檔案
- 內容包含：出勤天數、每日件數明細、職稱判定、各項加給、實領薪資

### 11. 職務加給欄位
- 薪資設定（後台基礎設定）新增「職務加給」
- 可針對特定員工設定固定每月加給金額
- 薪資計算時自動帶入，並顯示在薪資單上
